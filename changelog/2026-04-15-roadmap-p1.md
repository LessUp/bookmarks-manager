# Roadmap P1 功能实现记录

## 日期: 2026-04-15

## 完成功能

### P1-1: 多格式导出 ✅

实现了 JSON、CSV、Markdown 三种新的导出格式，支持在"上传合并"和"搜索"页面使用。

**新增文件:**
- `src/utils/exporters/index.ts` - 统一导出器模块
- `src/utils/exporters/exporters.test.ts` - 26 个测试用例

**支持的格式:**

| 格式 | 说明 | 文件扩展名 |
|------|------|-----------|
| HTML | 标准 Netscape 书签格式，浏览器兼容 | .html |
| JSON | 结构化数据，包含元数据 | .json |
| CSV | 表格格式，可用 Excel 打开 | .csv |
| Markdown | 文档格式，适合阅读分享 | .md |

**功能特性:**
- HTML 导出支持目录结构保留/平铺两种模式
- JSON 导出支持包含/不包含元数据选项
- CSV 导出支持字段转义（逗号、引号、换行）
- Markdown 导出自动生成目录层级标题
- 所有导出格式保持原有目录归一化逻辑

**界面更新:**
- 上传合并页面增加格式切换按钮组
- 搜索页面增加导出格式下拉选择器
- 导出文件名自动包含格式扩展名

## 技术实现

### 导出器设计
```typescript
export type ExportFormat = 'html' | 'json' | 'csv' | 'markdown'

export interface ExportOptions {
  preserveFolders?: boolean
  includeMetadata?: boolean
  prettyPrint?: boolean
  csvDelimiter?: string
}

// 统一导出入口
export function exportBookmarks(
  items: Bookmark[],
  format: ExportFormat,
  options?: ExportOptions
): string
```

### 备份服务设计
```typescript
export interface BackupData {
  version: number
  exportedAt: string
  bookmarks: StoredBookmark[]
  aiConfig?: AIConfig
  aiCache: AICache[]
  aiUsage: AIUsage[]
  aiPrompts: AIPrompt[]
  aiUsageLimits?: AIUsageLimits
  cleanupSessions: CleanupSessionRecord[]
}

// 创建备份
export async function createBackup(options?: BackupOptions): Promise<BackupData>

// 恢复备份
export async function restoreFromBackup(
  backup: BackupData,
  strategy: 'merge' | 'replace'
): Promise<RestoreResult>
```
