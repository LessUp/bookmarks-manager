## What's New in v1.1.0

### ✨ New Features

#### Multi-Format Export
- Export bookmarks as **JSON**, **CSV**, or **Markdown**
- HTML export with options for preserving or flattening folder structure
- CSV export with proper field escaping for complex data

#### Backup & Restore
- Complete application data backup including bookmarks, AI configuration, and usage records
- Selective backup options to reduce file size
- Preview statistics before restoring

#### Performance Optimizations
- **Web Worker** support for large datasets (>500 bookmarks)
- **Virtual scrolling** for search results (>200 items)
- ECharts on-demand loading reduced bundle size by 48.7%

### 📝 Documentation Improvements

- **Completely restructured documentation**
- New bilingual (English/Chinese) documentation:
  - Architecture documentation
  - API reference
  - Contributing guidelines
  - Product requirements document
- All docs now available in both languages

### 🔧 Improvements

- Enhanced search with 200ms debounce
- Improved file import error handling
- Better memory management with cleanup functions

## v1.1.0 更新内容

### ✨ 新功能

#### 多格式导出
- 支持将书签导出为 **JSON**、**CSV** 或 **Markdown**
- HTML 导出可选择保留或平铺文件夹结构
- CSV 导出支持复杂数据的字段转义

#### 备份与恢复
- 完整应用数据备份，包括书签、AI 配置和用量记录
- 选择性备份选项，减少文件大小
- 恢复前预览统计数据

#### 性能优化
- **Web Worker** 支持大数据集（>500 书签）
- **虚拟滚动** 搜索结果（>200 条）
- ECharts 按需加载，包体积减少 48.7%

### 📝 文档改进

- **全面重构文档结构**
- 新增双语（中英文）文档：
  - 架构文档
  - API 参考
  - 贡献指南
  - 产品需求文档
- 所有文档现支持中英文双语

### 🔧 改进

- 搜索添加 200ms 防抖
- 改进文件导入错误处理
- 通过清理函数优化内存管理
